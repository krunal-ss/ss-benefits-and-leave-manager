# PRD -- Employee Productivity Enhancements (v1.3)

Features: 1. Expense Receipt Preview 2. Leave Balance History 3.
Notification Preferences

Expense Receipt Preview: Preview uploaded receipts, zoom, rotate,
replace before submission.

Leave Balance History: Show opening balance, accruals, deductions,
adjustments, carry-forward, export.

Notification Preferences: Configure email, browser push, in-app
notifications and quiet hours.

Benefits: Better transparency, improved usability, personalized
notifications.
